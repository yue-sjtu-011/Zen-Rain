# Zen-Rain
Zen Rain | 禅意雨落

这是一个基于 MediaPipe 和 OpenCV 的手势交互项目。
灵感来源于极简主义插画，通过自拍视角识别手部动作，实现“五指张开，雨落涟漪”的视觉效果。

## 视觉特色
- **长雨丝设计**：模拟插画风的舒展线条。
- **多重涟漪**：具有随机偏移和不同心美感的同心圆环。
- **丝滑淡出**：涟漪随半径扩大逐渐归于背景色。

## 技术栈
- Python 3.11 (推荐)
- MediaPipe 0.10.14
- OpenCV

## 如何运行
1. 克隆或下载本项目。
2. 创建虚拟环境并激活。
3. 运行 `pip install -r requirements.txt` 安装依赖。
4. 运行 `python main.py`。
